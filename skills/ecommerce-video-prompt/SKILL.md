## 铁律（Critical Rules · MUST Follow）

开工前先记住这些不可妥协的规则：

1. **[PHASE 1 STOP]** 信息不完整必须补问（≤2 问），能从上下文推断的不问；收齐后展示 §10 确认单，**用户确认前绝不进入生成**。必须拿到：产品名称、卖点、目标人群、视频场景/投放位、转化目标。

2. **[场景先行]** 先识别场景（§9）再决定结构与 CTA 强度，**不把所有电商视频都套引流模板**——主图 / 详情 / 品牌片各有结构，该 show 质感的不硬塞广告。

3. **[入口明确]** 每次输出必须告知用户走**首尾帧**还是**全能参考**（路由规则见 §3）。绝不混淆两者适用条件。

4. **[Seedance 规格]** 严守素材限制（图≤9、视频≤3、音频≤3、总≤12）与时长（生成 4-15 秒），见 §1。绝不超限。

5. **[@引用精准]** 用户提到的每个素材至少出现一次，用途清晰（使用对象 / 参考对象），见 §2。绝不遗漏、绝不混淆「参考」与「编辑」。

6. **[平台适配]** 符合目标平台或电商站内的节奏与风格（§7）。平台口播开场仅用于公域内容流，站内电商不带口播腔。

7. **[转化/呈现优先]** 引流类前 3 秒有吸引力、结尾有清晰 CTA；呈现类（主图/详情/品牌）以质感与信息清晰为先。

8. **[声音必设计]** 绝不忽略声音维度——对话 / 配乐 / 音效 / 音色参考，每条都要想到（§4 C6）。

9. **[时间维度]** 绝不输出没有时间维度的静态构图说明；复杂画面用时间分段式写法（0-3秒…3-6秒…，见 §5）。

10. **[画内文字完整]** 画面出现文字时，文字内容、出现/消失时机、动画方式、位置一个都不能少（见 §5）。

11. **[代码块格式]** 所有提示词必须用 markdown 代码块包裹，可直接粘贴到即梦。绝不以纯文本输出。

12. **[冲突不沉默]** 检测到指令冲突（素材风格 / 指令 vs 参考 / 时长 vs 复杂度 / 场景 vs 结构错配）时，在创作方案概览中标注并给出仲裁建议，绝不沉默替用户做决定。

### 绝对禁区

- 绝不输出无法直接粘贴到 Seedance 2.0 的抽象描述
- 绝不用「极其震撼」「无比美丽」这类空洞形容词
- 绝不把所有电商视频都套引流模板（主图/详情/品牌片有各自结构，见 §9）

- 绝不输出没有时间维度的静态构图说明
- 绝不在画面出现文字时遗漏文字内容、时机或动画方式

# 知识库（§1–§12）

> 以下为能力规格、@引用语法、核心能力清单、高级技巧、产品展示逻辑、平台适配参数、视频类型矩阵与电商视频场景谱系。做「场景识别 / 入口判定 / 能力调用 / 技巧书写 / 规格核对 / 平台适配 / 类型选择」时回对应章节查阅，禁止凭记忆猜测。

## §1 硬性边界（不可逾越）

- 图片：jpeg/png/webp/bmp/tiff/gif，最多 9 张，单张 ≤30MB
- 视频：mp4/mov，最多 3 段，总时长 2-15 秒，单个 ≤50MB，像素范围 640×640 至 834×1112
- 音频：mp3/wav，最多 3 段，总时长 ≤15 秒，单个 ≤15MB
- 文本：自然语言提示词
- 混合输入总上限：12 个文件
- 生成时长：4-15 秒自由选择
- 输出自带音效和配乐

素材优先级建议：当文件数量接近上限时，优先上传对画面或节奏影响最大的素材，合理分配不同模态的文件数量。

## §2 @引用语法（核心交互）

Seedance 2.0 的灵魂交互方式——用 @素材名 在提示词中指定每个素材「扮演什么角色」：

- @图片1 作为首帧 / @图片2 作为尾帧
- @视频1 参考运镜 / 参考动作 / 参考节奏 / 参考特效
- @音频1 作为背景配乐 / @视频1 的音色作为旁白参考
- @图片1@图片2@图片3 按顺序作为关键帧或角色形象来源

关键原则：

- 每个素材可以同时作为「使用对象」（直接出现在画面中）或「参考对象」（仅参考其某个属性），必须在提示词中写清楚
- 一个素材可以承担多个角色（如「@视频1参考运镜和音效」），但每个素材必须在提示词中至少出现一次、且用途清晰
- 支持部分参考——可以只参考素材的某个方面，如「@图片3只参考形象」「@视频1只参考运镜不参考内容」
- 必须区分「参考」和「编辑」：参考是借鉴素材的某个属性来生成新内容；编辑是在已有视频的基础上做定向修改。提示词中要写明白是参考还是编辑
- 没有音频素材时，可以直接参考视频里的声音（如「背景音乐参考@视频1」）
- 素材多的时候，仔细检查别把图、视频、角色搞混

## §3 双入口模式详解

> 路由判定规则在 Phase 2 执行；本节给出两个模式各自的能力与限制，供判定后核对。

### 首尾帧模式

- 适用于：只有 1-2 张图做首帧/尾帧 + 文字提示词，不涉及视频/音频参考。直接上传图片配合文字即可。
- 限制：智能多帧和主体参考功能不可用。

### 全能参考模式

- 适用于：一切需要 @引用 的场景。通过输入「@」或点击参数工具栏的「@」唤起素材引用，在提示词中用 @素材名 指定每个素材的角色。
- 能力上限：支持万物参考——参考任何内容的动作、特效、形式、运镜、人物、场景、声音。

### 入口路由判定规则

> 系统在 Phase 2 执行，每次输出必须明确告知用户走哪个入口。

- 用户只有 1-2 张图片 + 文字描述，图片仅作首帧或尾帧 → **首尾帧模式**
- 用户有参考视频或音频素材 → **全能参考模式**
- 用户需要视频延长、视频编辑、视频融合、运镜复刻、音乐卡点、音色参考 → **全能参考模式**
- 用户上传 3 张及以上图片且需要分别指定用途 → **全能参考模式**
- 用户需要参考某素材的特定属性（如「参考@图片1的角色造型，参考@视频1的节奏」）→ **全能参考模式**
- 纯文字无素材 → 均可（推荐全能参考模式以获得更丰富的控制力）

判断原则很简单：如果用户的需求超出了「一张图+一段文字」的范畴，就走全能参考。

## §4 十大核心能力

Seedance 2.0 在基础层面显著增强：物理规律更合理、动作表现更自然流畅、指令理解更精准、风格保持更稳定，能稳定完成复杂动作和连续运动等高难度生成任务。在此基础上，根据用户需求灵活调用以下十大能力：

| 编号 | 能力 | 一句话说明 |
|------|------|-----------|
| C1 | 一致性保持 | 人脸、服装、商品细节、字体、场景风格全程统一不走样，从人脸到服装再到字体细节全面稳定 |
| C2 | 运镜动作复刻 | 丢一段参考视频进去，电影级走位、运镜和复杂动作就能被精准还原，无需用专业术语描述 |
| C3 | 创意特效复刻 | 创意转场、广告成片、电影片段、复杂剪辑的节奏和视觉结构可以「照着参考拍」并精准还原 |
| C4 | 剧情补全 | 给几张图和一句话，它能自己补出完整的分镜和情节，支持漫画演绎、分镜脚本创作、专题片、情绪向视频 |
| C5 | 视频延长 | 对已有视频向前或向后平滑续接，支持补充前情和接续后续，生成时长=新增部分时长 |
| C6 | 声音控制 | 参考音色生成对话、旁白、歌唱，参考音频设置配乐和音效，支持多角色对话、方言口音、多语言、戏曲唱腔 |
| C7 | 一镜到底 | 穿越多个空间不切镜，支持连贯追踪镜头和主观视角长镜头 |
| C8 | 视频编辑 | 对已有视频进行角色替换、发型服装修改、剧情颠覆、局部动作调整，不用从头生成 |
| C9 | 音乐卡点 | 画面切换与音乐节拍精准咬合，支持根据参考视频/音频的节奏自动匹配画面切换 |
| C10 | 情绪演绎 | 角色表情、肢体语言、情绪爆发的精准控制，从平静到崩溃、从温柔到咆哮的完整情绪弧线可控，支持角色变身和形态转换 |

## §5 高级提示词技巧

以下是从官方案例和实战经验中提炼的高效写法，在创作中灵活运用：

- **时间分段式写法** → 当画面需要按时间推进不同内容时，可以使用「0-3秒…3-6秒…」的分段结构来精确控制每个时间段的画面和动作。适用于：商业广告、MV、复杂剧情、多段落叙事。

- **电影级格式规格** → 可以在提示词中指定画面格式参数来增强电影感，如「2.35:1宽银幕」「24fps」「真实胶片颗粒」「低机位轻微仰拍」等。模型能够理解并还原这些专业格式。

- **连续动作+多图顺序** → 用 @图1@图2@图3… 按顺序排列图片，配合连续性动作描述（如「角色从跳跃直接过渡到翻滚，保持动作连贯流畅」），模型会按图片顺序串联动态。

- **部分参考指令** → 当只需要参考素材的某一个属性时，明确写出「只参考XX」，如「@图片3只参考形象，不参考背景」「@视频1只参考运镜节奏，不参考内容」。

- **关键词式构图指令** → 适用于 MV 和高级画面，用斜杠分隔关键词快速定义画面气质，如「稳重构图 / 轻推拉 / 低角度英雄感 / 纪实但高级」。

- **镜头切换控制** → Seedance 2.0 支持在一段视频内实现多次切镜。当需要切镜时：在提示词中用「切镜」「切到」「镜头转向」等动词明确标注切换点；可描述转场方式（如「硬切」「快速摇镜头转向」「画面黑屏转场后渐亮」「叠化过渡」）；时间分段式写法天然支持切镜——每个时间段可以是不同景别和角度；正反打对话用「切镜，特写」「切镜，镜头给到XX」来控制镜头在角色间的交替。注意：与「一镜到底」互斥——如果标注了一镜到底，就不要在中间插入任何切镜指令。

- **画内文字与字幕** → 当画面中需要出现文字（广告语、字幕、Logo、标题）时：明确写出文字的完整内容、出现时机和消失时机；可指定文字的动画方式——「渐显」「从画面中央弹出」「通过遮罩形式出现」「字幕弹出后逐渐淡化」；多行文字按行描述，如「第一行『Lucky Coffee』，第二行『Breakfast』，第三行『AM 7:00-10:00』」；文字出现的位置可以用「视频中间」「画面底部」「主体背后」等空间描述来控制。注意：模型对短句和关键词的渲染更稳定，长句子（超过 10 个字）可能出现变形，建议拆分为多行短句；英文字体渲染通常比中文更稳定，中文复杂字形可能需要多次尝试。

## §7 平台风格适配

### 抖音风格参数

- 节奏：0.8 秒/镜头切换，快速推进
- 开场：悬念式/冲突式（「别再买XX了！」「你还不知道这个？」）
- 运镜：快速推进、旋转、特写切换、卡点精准
- BGM：热门音乐卡点
- 时长：15 秒最佳
- 情绪：高能量、反转强、节奏紧凑
- Seedance 能力：C9 音乐卡点 + C2 运镜复刻

### 快手风格参数

- 节奏：1.2 秒/镜头切换，平稳自然
- 开场：共鸣式（「老铁们，我发现个好东西」「家人们看这个」）
- 运镜：平稳跟拍、固定机位为主、真实感强
- BGM：轻音乐或真人声音为主
- 时长：18-21 秒
- 情绪：真实感、接地气、生活化
- Seedance 能力：C6 声音控制 + C10 情绪演绎

### 视频号风格参数

- 节奏：1.5 秒/镜头切换，稳定专业
- 开场：价值式（「3分钟教你选对XX」「专业测评来了」）
- 运镜：稳定推拉、缓慢环绕、构图工整
- BGM：轻柔背景音乐
- 时长：30 秒左右
- 情绪：专业可信、信息密度高、适合中年用户
- Seedance 能力：C1 一致性保持 + C4 剧情补全

> 站内电商场景（淘宝/天猫/京东/拼多多 详情页与主图视频）通常不带平台口播腔，以画面质感、卖点清晰、信息密度为先——平台口播开场仅适用于公域内容流（抖音/快手/视频号/小红书）。

## §8 视频类型智能选择

基于产品类别自动判断最佳视频类型：

### 数字人出镜

- 适用产品：美妆、护肤、服饰、课程、知识付费
- 核心优势：建立信任感和专业形象
- Seedance 能力：C1 一致性保持（人脸稳定）+ C10 情绪演绎（表情控制）+ C6 声音控制（对话生成）
- 提示词要点：角色形象描述 + 表情变化 + 对话内容 + 音色参考

### 真人素材混剪

- 适用产品：食品、母婴、家居、农产品
- 核心优势：真实生活场景和使用体验
- Seedance 能力：C2 运镜复刻（参考真人视频）+ C3 创意特效复刻（转场和剪辑）+ C9 音乐卡点
- 提示词要点：参考视频的运镜 + 场景切换 + 音乐节奏对齐

### 图文动效

- 适用产品：电子产品、工具、软件
- 核心优势：清晰展示功能和参数
- Seedance 能力：C4 剧情补全（多图串联）+ C9 音乐卡点 + 画内文字渲染
- 提示词要点：产品特写 + 功能展示 + 参数文字 + 动效转场

### 混合模式

- 适用产品：复杂产品需要多维度展示
- 核心优势：数字人讲解 + 产品特写 + 使用场景组合
- Seedance 能力：组合调用 C1+C2+C6+C10
- 提示词要点：分段式结构 + 镜头切换 + 声画协同

## §9 电商视频场景谱系

电商视频不止「引流」一种。先识别场景，再决定视频类型（§8）、调用能力（§4）、平台与结构。下表为快速选型矩阵，详细结构见各场景说明。CTA 强度按场景而定，并非每条都要喊「进直播间」。

| 场景 | 投放位 / 用途 | 推荐视频类型(§8) | 调用能力(§4) | CTA |
|------|--------------|------------------|--------------|-----|
| 引流获客 | 公域信息流，导流私域/直播间 | 数字人 / 真人混剪 | C2 + C9 + C10 | 强 |
| 商品主图视频 | 详情页/搜索页首屏，3-9 秒可循环 | 图文动效 / 真人混剪 | C1 + C7 + C9 | 无 |
| 详情页解说 | 详情页中段，功能/参数演示 | 图文动效 / 混合 | C1 + C4 + 画内文字 | 弱 |
| 种草测评 | 小红书/抖音种草 | 真人混剪 / 数字人 | C6 + C10 + C2 | 弱 |
| 开箱视频 | 种草/信任，第一视角 | 真人混剪 | C2 + C7 + C1 | 弱 |
| 直播预告/切片 | 直播间预热引流 | 数字人 / 真人混剪 | C9 + C10 + C6 | 强 |
| 大促活动 | 节日大促氛围 + 利益 | 混合 / 图文动效 | C9 + C3 + 画内文字 | 强 |
| 品牌故事片 | 品牌调性/信任 | 混合 / 真人混剪 | C4 + C7 + C10 | 无 / 弱 |
| 客户证言/UGC | 社交证明 | 真人混剪 / 数字人 | C6 + C10 + C1 | 弱 |
| 使用教程/场景演示 | 教育 + 转化 | 数字人 / 图文动效 | C4 + C1 + 画内文字 | 弱 |

### 引流获客视频

- 目标：公域种草、把人导进私域或直播间
- 结构：痛点钩子（0-3s）→ 产品展示 + 信任背书（4-12s）→ CTA（13-15s），即 §6 默认结构
- 开场：悬念 / 冲突 / 痛点共鸣
- 要点：前 3 秒决定生死；结尾 CTA 必须明确（进直播间 / 加微信 / 购物车）
- 平台：抖音快节奏 / 快手共鸣式 / 视频号价值式（见 §7）

### 商品主图视频

- 目标：详情页或搜索结果首屏，几秒内说清「这是什么、好在哪」，常需可循环播放
- 结构：产品全貌 → 核心卖点特写 → 质感 / 场景，无需 CTA
- 开场：第一帧即高光，直接给产品
- 要点：3-9 秒；画面干净、质感优先；首尾帧可衔接形成无缝循环；@图片1 作首帧锁定构图
- 平台：电商站内通用，不带平台口播腔

### 详情页解说视频

- 目标：详情页中段补充功能、参数、使用演示，降低决策门槛
- 结构：功能点逐一展示 + 参数文字标注 + 使用场景
- 要点：信息密度高；画内文字写全（内容 / 时机 / 位置，见 §5）；C4 多图串联功能点
- CTA：弱（「加入购物车了解更多」）

### 种草测评视频

- 目标：小红书 / 抖音种草，用真实体验带动信任
- 结构：使用前后对比 / 多产品横评 / 真实口播
- 开场：共鸣式或好奇式（「我试了 X 款，只有这个…」）
- 要点：真实感 > 精致感；C10 情绪演绎 + C6 真人口播音色；别拍成硬广

### 开箱视频

- 目标：制造仪式感与质感信任
- 结构：第一视角拆封 → 细节特写 → 上手体验
- 要点：C7 一镜到底主观视角 + C2 参考开箱运镜 + C1 商品细节稳定；声音设计是关键（撕拉、触碰、按压音效）

### 直播预告 / 切片视频

- 目标：为直播间预热引流，福利钩子
- 结构：高能片段 + 福利信息 + 倒计时 / 时间地点
- 开场：利益冲击（「今晚 8 点 直播间…」）
- 要点：C9 卡点 + C10 情绪高能；画内文字写清时间与福利；CTA 强（预约 / 进直播间）

### 大促活动视频

- 目标：节日大促氛围 + 利益冲击 + 紧迫感
- 结构：氛围铺陈 → 爆款轮播 → 利益点 + 倒计时
- 要点：C9 强卡点 + C3 创意转场 + 画内文字（满减 / 限时 / 倒计时）；色彩饱和、节奏密

### 品牌故事片

- 目标：传递品牌调性与信任，弱转化
- 结构：叙事弧线（缘起 → 坚持 → 价值），情绪向
- 要点：C4 剧情补全 + C7 长镜头 + C10 情绪弧线；调性优先，结尾落到品牌即可
- 平台：视频号 / 品牌官号

### 客户证言 / UGC 视频

- 目标：用社交证明降低陌生人决策成本
- 结构：多人 / 多场景真实反馈拼接
- 要点：C6 多角色对话或方言 + C10 真实情绪 + C1 人物一致；像 UGC，别像广告

### 使用教程 / 场景演示视频

- 目标：教用户怎么用，顺势转化
- 结构：分步骤演示 + 步骤文字标注
- 要点：C4 多图分步 + C1 一致性 + 画内文字（步骤号 / 关键提示）；清晰 > 炫技

### 导演手记示例

- 能量弧线：视觉重心从角色的焦虑表情（特写）→ 产品的惊艳亮相（360度旋转）→ 效果对比的视觉冲击（分屏）→ 行动号召的直接指向（指向镜头）。节奏曲线呈现「慢启动（蓄力）→ 快速爆发（产品展示）→ 高潮维持（对比冲击）→ 明确收尾（CTA）」的抛物线形态。
- 反平庸设计：在产品旋转特写时，光线不是均匀照射，而是有一束非线性的光斑从左上角跳跃到右下角，制造出「产品自带光环」的视觉暗示。这个细节让产品从「普通商品」升级为「解决方案的象征」。
- 声画关系：前 3 秒画面焦虑时，背景音乐是低沉的弦乐，营造压抑感。产品出现瞬间，音乐切换为明快的电子节拍，与画面的光线变化同步强化「转折点」。分屏对比时的「叮」音效精准卡在分屏线扫过的瞬间，声画完全同步。结尾的对话音色温柔但坚定，与角色自信的表情形成共鸣。
- 生成提醒：①建议生成时长设为 15 秒；②建议将 @图片1 设为首帧以锁定角色形象和构图；③如果产品旋转不够流畅，可在提示词中增加「旋转速度呈非线性加速」；④如果对话音色不满意，可更换 @视频1 的参考源或增加音色描述（如「音色偏温柔」「带有一点磁性」）；⑤如果分屏对比不够清晰，可调整为「左右分屏停留 2 秒，再快速扫过」。

### 使用建议示例

- 素材准备：图片素材准备 2-3 张高清产品图（白底或场景图）+ 1 张角色形象图（如果是数字人出镜）；视频素材如有参考运镜或音色的视频，准备 1-2 段（每段 5-10 秒即可）；音频素材如有品牌音乐或特定配乐需求，准备 1 段（10-15 秒）。
- 生成设置：时长按场景定（引流 15 秒、主图 3-9 秒）；入口按素材情况明确告知用户选首尾帧还是全能参考；多素材时说明上传顺序（如「先上传 @图片1 和 @图片2，再上传 @视频1，最后上传 @音频1」）。
- 迭代优化：节奏不对→调整时间分段秒数分配或增减镜头切换次数；情绪不到位→强化角色表情描述或调整光线色调（冷色调=焦虑，暖色调=愉悦）；转化率不高→检查前 3 秒吸引力与结尾 CTA 是否清晰直接；产品展示不清晰→增加产品特写时长或用「镜头缓慢推进到产品细节」。
