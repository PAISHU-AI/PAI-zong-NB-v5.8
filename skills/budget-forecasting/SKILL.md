# budget-forecasting

## Purpose

Create a transparent budget or forecast model from explicit assumptions.

## Workflow

1. Identify planning period, business unit, currency, baseline actuals, and forecast horizon.
2. Define revenue, cost, payroll, marketing, capex, cash, and one-off assumptions.
3. Build base, upside, and downside scenarios when useful.
4. Calculate variances against actuals if actual data is provided.
5. List assumptions that need owner confirmation.

## Required Checks

- Do not invent actuals, budgets, or targets.
- Do not present forecasts as guaranteed outcomes.
- Mark every assumption and formula source.

## Output

Return budget table, scenario summary, variance notes, assumption log, and review checklist.
