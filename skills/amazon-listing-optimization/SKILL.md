# amazon-listing-optimization

## Purpose

Create or improve Amazon listing content that helps shoppers understand the product, compare options, and buy with confidence.

## Workflow

1. Identify product, category, target shopper, claims, proof, objections, and compliance limits.
2. Draft title, bullets, description, and search-term themes.
3. Map each image slot to a selling job: hero, scale, use case, proof, comparison, bundle, or FAQ.
4. Add conversion notes for mobile readability and marketplace scanning.
5. Mark assumptions and claims that require seller verification.

## Required Checks

- Do not invent certifications, rankings, reviews, guarantees, or performance claims.
- Keep restricted-category or medical-like claims conservative.
- Do not imply Seller Central changes were made.

## Output

Return optimized listing copy, image-slot map, keyword themes, claim checklist, and revision notes.
