# ecommerce-product-description

## Purpose

Create conversion-oriented product listing copy that balances search intent, buyer objections, and truthful claims.

## Workflow

1. Capture product facts, target buyer, platform, differentiators, objections, proof, and forbidden claims.
2. Identify search terms, buying scenarios, and comparison angles.
3. Write title, bullets, description, and optional backend keywords.
4. Add variants by platform when needed.
5. Provide improvement notes for images, reviews, FAQs, and A+ or rich content.

## Required Checks

- Do not fabricate certifications, materials, guarantees, reviews, or performance claims.
- Keep claims platform-compliant and evidence-backed.
- Prefer specific benefits over generic adjectives.

## Output

Return title options, bullet set, full description, keywords, FAQ ideas, and optimization checklist.
