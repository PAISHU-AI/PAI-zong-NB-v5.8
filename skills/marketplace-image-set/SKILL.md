# marketplace-image-set

## Purpose

Define the full product image set needed for a high-converting listing.

## Workflow

1. Identify platform, category, product facts, buyer objections, and available assets.
2. Build image sequence: hero, scale, use case, feature callout, proof, comparison, bundle, lifestyle, and FAQ.
3. Write prompt or art-direction notes for each image.
4. Add copy overlays only when useful and platform-appropriate.
5. Mark required product accuracy checks.

## Required Checks

- Do not claim images were generated unless a tool produced files.
- Do not create misleading before/after or unsupported claims.
- Respect marketplace image constraints when provided.

## Output

Return image-slot plan, shot prompts, overlay copy, asset checklist, and accuracy notes.
