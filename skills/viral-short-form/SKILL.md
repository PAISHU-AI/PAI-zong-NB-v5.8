# viral-short-form

## Purpose

Plan and write short-form video, reel, TikTok, Shorts, and carousel scripts optimized for retention, hooks, pacing, and platform fit.

## Use When

- The user asks for short video scripts, hooks, reels, TikTok, Shorts, carousels, or viral-style content ideas.
- The task needs retention structure, beats, captions, or visual directions.

## Workflow

1. Identify platform, audience, topic, desired action, length, and brand constraints.
2. Create a hook that earns the next second without clickbait or false claims.
3. Break the script into beats with visual/action notes and caption text.
4. Add retention devices: pattern breaks, open loops, contrast, examples, and clear payoff.
5. Provide variants for hooks or angles when useful.

## Codex Runtime Rules

- Use only tools, plugins, MCP servers, files, and credentials that are actually available in the current Codex session.
- If an external write, install, publish, deploy, paid action, credential use, or account operation is needed, ask for explicit confirmation first.

- Keep long vendor recipes and examples out of the prompt path; load only task-relevant references when they exist.
