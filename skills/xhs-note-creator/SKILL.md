# xhs-note-creator

## Purpose

Turn source material into a complete Xiaohongshu note package that can be used for copy, card design, and publishing preparation.

## Workflow

1. Clarify topic, target reader, note goal, source material, and product or experience facts.
2. Draft the note: titles, body, closing, and tag set.
3. Create card-ready Markdown with a cover section and inner sections.
4. Split long material into card pages with one idea per card.
5. Provide cover visual direction, inner-card hierarchy, and image prompts when generation tools are available.
6. Add publishing checklist and risk notes.

## Required Checks

- Keep card text concise; each card should have one main point.
- Preserve user-provided facts and mark assumptions.
- Do not claim actual publishing, account access, or generated images unless tool evidence exists.
- For commercial products, separate real experience from marketing claims.

## Output

Deliver a note package: title options, body, tags, card Markdown, cover prompt, inner-card prompts, and publish checklist.
