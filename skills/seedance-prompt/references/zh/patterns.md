# Seedance 2.0 — 场景模式（中文）

> 按需加载模块。硬性上限与流程见 SKILL.md。

每个模式给出一个常见形态和一段原创示例。措辞可改，结构照搬。

### 1. 跨镜头保持同一人物
把外观锚到一张图，让动作自然走。
```
@图片1 的小提琴手在侧幕等待，闭着眼，松了松肩。特写她的手——
拧紧弦轴，试一下音。她吐气，睁眼，神情安静而笃定，迈向舞台
灯光。后台暖光，远处隐约的观众声。
```

### 2. 沿用一段运镜
把参考片的运动逐帧搬过来。
```
主角用 @图片1 的滑板少年，在 @图片2 的水泥涵洞里。逐帧沿用
@视频1 的运镜路径——贴着板面低位滑行跟拍，他把板一踢的瞬间
甩镜。表情专注、近乎漠然。轮子碾过水泥的回声。
```

### 3. 复刻创意模板 / 特效
把主体换进参考片的特效语言里。
```
把 @视频1 的主体换成 @图片1，以 @图片1 开场。自由潜水员从崖边
蹬出。完全套用 @视频1 的转场——镜头跟她下坠，破开水面进入
@图片2 的珊瑚礁。她穿过一道海藻门，进入 @图片3 的沉船船体，
光尘在身边洒落。全程闷住的水下静音感。
```

### 4. 向后延长片段
续写已有视频，生成长度只设新增的那段。
```
将 @视频1 延长 15 秒。
1–5秒    唱针落下，黑胶转动，尘埃在午后斜光里飘。
6–10秒   镜头缓缓推向旋转的唱片标签，画面渐暗。
11–15秒  字幕渐显："NIGHT OWL RECORDS"，再一行"营业到深夜"。
```
向前延长则改成前置：
```
向前延长 10 秒：晨光爬上山脊，镜头缓缓下俯，落到一顶帐篷，
小火堆升起一缕烟。
```

### 5. 编辑已有片段
大部分保留，只改一件定好的事。

反转基调：
```
重做 @视频1 的气氛——面试官温和的笑慢慢转冷，变得平淡。他
一言不发，把一张纸推过桌面，往后一靠。应聘者的期待一点点
落空。不要抬高音量，只有纸张的摩擦和一段长长的沉默。
```
替换主体：
```
把 @视频1 里的跑步者换成 @图片1 的骑行者，路线和剪辑保持
完全一致，不要新增镜头。
```
添加元素：
```
@视频1 里人物的身后，一群孔明灯升空，飘进暮色天空。
```

### 6. 卡着节拍剪
把静帧同步到音轨。
```
@图片1 @图片2 @图片3 @图片4 @图片5 @图片6 —— 球鞋主图。按
@视频1 的关键帧和整体律动卡点。每帧加一点动态，推高对比，
落点处闪一道轮廓光。景别随节奏逐帧调整。
```

### 7. 对话与配音
把台词和演法都写出来。
```
《下班后的办公室绿植》—— 一段冷幽默双口相声。
仙人掌（无所谓，慢条斯理）："你又蔫了。说过多少次，少点
戏，少点水。"
蕨类（被冒犯，叶子沙沙）："你好意思说？从开春到现在你一毫米
都没长。"
冷面喜剧节奏，熄灯后柔和的办公室环境音。
```

### 8. 一镜到底
一个镜头，不切。
```
@图片1 @图片2 @图片3 @图片4 —— 一镜到底跟拍一名端着满托盘的
服务员，走出叮当作响的后厨，穿过备餐走廊，推开弹簧门，来到
黄昏时分的天台露台。全程不切。
```
带场景切换的一镜到底：
```
以 @图片1 开场。镜头跟拍一名外卖骑手骑摩托穿过狭窄老城巷子，
全景，摊位与行人不断穿过画面。到拐角处——@图片2 的拱门——
他刹住，把包裹递给一个参考 @图片3 的身影，随后镜头上摇，跟他
消失进 @图片4 的市集人群。一镜到底，全程不切。
```

### 9. 产品展示
让物件当主角。

拆解重组：
```
固定机位，柔和无缝背景。@图片1 的耳机充电盒悬浮、打开，两只
耳塞升起旋转。盒体分离成干净的分层组件——外壳、电池、主板——
在半空保持间距，随后一切滑回，合成一个密封的盒。流畅，不耍
花活，高级产品光，合盖时一声轻微的磁吸咔哒。
```
旋转揭示：
```
@图片1 的球鞋高速旋转两圈，急停，随即裂成鞋面、中底、外底
悬浮分开片刻，再砰地合回完整一只。3D 渲染质感，干脆的落点。
```

### 10. 科普 / 讲解
把一个概念讲干净。
```
15 秒水循环讲解。
0–5秒    一道山谷剖面；阳光晒暖河流，水汽缕缕升起聚成云。
5–10秒   云变厚变冷；雨落过山脊，水流顺着回到河里。
10–15秒  拉远展示完整闭环——海、天、雨、河——配简单箭头标注。
干净、平静、教学向的色调。
```

### 11. 漫改 / 微短剧
把一页或一组分镜演活。
```
把 @图片1 逐格演绎，从左到右、从上到下。每个人物的台词与
页面保持一致。分镜切换和关键节点配上利落音效；轻松诙谐的
基调。演法参考 @视频1。
```
按分镜生成片头：
```
以 @图片1 的分镜为蓝本——它的分格、景别、运镜和文案——做一段
15 秒、温柔治愈、题为《破晓的城市》的片头。
```

### 12. 视频融合
把几段视频接成一条流。
```
@视频1 里一滴墨在水中晕开；扩散时溶进 @视频2，一朵延时盛开
的花；花瓣散开又重组成 @视频3 缓慢旋转的星系，随后消散。
全程梦幻，配低沉的环境嗡鸣和轻柔风铃。
```

## 组合玩法（不是固定菜单）
- 首/尾帧 + 参考片的动作：`以 @图片1 开场，复刻 @视频1 的打斗动作`
- 延长后继续：`将 @视频1 延长 5 秒`（生成也选 5 秒）
- 桥接片段：`在 @视频1 和 @视频2 之间加一个场景，内容是……`
- 没音频文件但想要某种声音感：直接引用某段视频里的声音
- 串一个连续动作：`她从腾空直接进入翻滚，不间断` + `@图片1 @图片2 @图片3 …`
