# Seedance 2.0 — 模板库与风格修饰（中文）

> 按需加载模块。硬性上限与流程见 SKILL.md。

填空式起手模板。替换方括号内容和 `@` 素材即可。

### 产品广告片（15秒）
```
沿用 @视频1 的剪辑律动和转场。把它的产品换成 @图片1 当主角。
做一支 15 秒广告。
0–3秒    主角一个转身入场；特写做工与 logo。
4–8秒    多角度切换——正、侧、背——配一道扫过的高光。
9–12秒   使用场景：有人在真实环境里伸手去拿。
13–15秒  主视觉定帧，广告语落定，音乐收束。
音效：复用 @视频1 的底乐，加一声轻微的交互咔哒。
```

### 微短剧（15秒）
```
0–5秒    特写火车站检票口前的一张脸，在人群里搜寻，屏着呼吸。
        [台词A，小心翼翼]："我没想到你真会来。"
6–10秒   反打：另一人背着包，先别开眼，又转回来。镜头掠过站台
        时钟。[台词B，很轻]："我差点没来。"
11–15秒  两人走近，绷着的张力化成一个拥抱。镜头缓缓后退。
        柔和钢琴，底下垫着车站环境音。
精准 15 秒，每一拍都不浪费。
```

### 舞蹈 / 动作（13秒）
```
让 @图片1 的人物表演 @视频1 里的编舞和节拍。13 秒，动作干净
连贯，不卡顿不僵帧。
```

### 旅行风光卡点（15秒）
```
@图片1 @图片2 @图片3 @图片4 @图片5 —— 风光静帧。按 @视频1 的
节奏卡点，匹配它的转场感、色调和音乐速度。
```

### 仙侠 / 奇幻（15秒）
```
15 秒高燃仙侠，冷蓝与银的色调。
0–3秒    低角度仰拍：剑仙立于浮空山巅，衣袍在风里猎猎，手中一柄
        霜蓝长剑低鸣。她低语："此门尚在。"风声与一缕剑鸣。
4–8秒    环绕快切，她横扫长剑，道道苍白火光把影兽击散成雾。
        剑鸣与嘶嚎。
9–12秒   升降臂上摇、慢动作：她跃起，剑刃凝出一圈凤凰之火劈下，
        山巅在她脚下裂开光芒。
13–15秒  缓推，她落地收剑，衣袍渐定，目光清冷。定格在远处一道
        云中之门；声音收束为风声。
```

### 科普 CG（15秒）
```
干净的 4K 图解风格，平静的教学色调。发射台上一枚火箭；镜头
推近，发动机点火。中段切到一张简洁剖面图：分级、燃料流向、
一级分离，配文字箭头标注。最后近景，舱体抵达轨道，地球弧线
在下方。稳健、教学向。15 秒讲解。
```

## 风格修饰（贴在末尾提质）

**观感**——"电影感，胶片颗粒，浅景深" · "2.35:1，24fps" · "水墨" / "动漫" / "超写实" · "霓虹色调，冷暖分割" · "干净 4K 渲染，半透明" · "精细打磨的 CG"。

**情绪**——"紧张，屏息" · "温暖治愈" · "史诗恢宏" · "喜剧，表情夸张" · "纪录片，克制" · "暗黑奇幻" / "高燃仙侠"。

**声音**——"配乐：开阔激荡" · "音效：脚步、人群、车流" · "旁白音色照 @视频1" · "剪辑卡死在节拍上" · "脚步、呼吸、衣料摩擦清晰且贴拍"。
