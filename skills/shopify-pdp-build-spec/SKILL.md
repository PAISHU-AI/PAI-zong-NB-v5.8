# shopify-pdp-build-spec

## Purpose

Translate ecommerce PDP strategy into a Shopify-ready build specification for design or implementation.

## Workflow

1. Identify theme, product type, page sections, variants, metafields, media, reviews, and offer blocks.
2. Define section order and required content for each block.
3. Map dynamic fields, variant behavior, image/video slots, FAQ, trust blocks, and cross-sells.
4. Add QA for mobile, accessibility, price/variant clarity, and page speed.
5. Separate strategy from code implementation.

## Required Checks

- Do not assume a Shopify app or theme feature exists without evidence.
- Do not modify store files unless the user asks for implementation.
- Keep account/admin actions gated.

## Output

Return build spec, metafield map, content requirements, QA checklist, and implementation notes.
