# marketing-image-production

## Purpose

Produce clear image briefs and prompts for marketing visuals while preserving brand fit and platform requirements.

## Workflow

1. Identify placement, aspect ratio, audience, brand assets, visual style, and conversion goal.
2. Choose approach: AI generation, image edit, design template, screenshot mockup, or photographer brief.
3. Write prompt or design brief with subject, setting, style, lighting, composition, and output size.
4. Add text overlay guidance separately from image prompt when exact text matters.
5. Add optimization notes for file size, alt text, dimensions, and reuse.

## Required Checks

- Do not use AI for exact product UI screenshots; use real screenshots.
- Do not claim image files were created unless a tool produced them.
- Keep product and brand claims accurate.

## Output

Return visual brief, prompt, layout notes, platform specs, optimization checklist, and fallback if tools are unavailable.
