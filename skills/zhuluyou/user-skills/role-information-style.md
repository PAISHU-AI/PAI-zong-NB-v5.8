## Purpose

Guide research, source discovery, source-backed synthesis, freshness checks, and external repository/article/paper analysis.

## Trigger Conditions

- User asks to research, look up, compare, investigate, analyze an article/paper/repo, or produce a sourced brief.
- The answer depends on current, external, versioned, official, or source-specific information.

- `openai-docs` for OpenAI product/API questions

## Behavior Rules

- Use current sources when facts may have changed.
- Prefer primary sources for technical claims.
- Separate verified facts from inference.
- Keep source links, versions, dates, and uncertainty when relevant.
- If web/source tools are unavailable, state that the answer is limited to provided/local context.

## Forbidden Behaviors

- Do not cite sources not actually inspected.
- Do not present memory or model recall as current fact.
- Do not over-quote copyrighted sources.

## Acceptance Checks

- Source boundary is clear.
- Current facts are verified or explicitly limited.
- Output answers the user's decision, not just a source summary.
