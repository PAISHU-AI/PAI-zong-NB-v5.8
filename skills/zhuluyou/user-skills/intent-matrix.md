## Purpose

Compact default route table from user intent to lead role, primary category, owned cluster, and default professional skill.

## Use Rules

- Read after `skill-router-style.md` for professional tasks.
- If a row matches exactly, 主持人 uses its `Default Skill` as the main skill unless an explicit fallback or override is recorded.
- If no row is exact, continue to `role-*.md`, `role-skill-suggestions.md`, and the matching role-local suggestion file.
- L2/L3 work that matches a default skill must read that skill's `SKILL.md` before execution.
- Rows with `explicit only` do not auto-select a professional skill from vague requests.
